# GitHub Actions Cloud Monitoring Guide

## How It Works
1. GitHub automatically runs your monitor every 5 minutes
2. Results are saved as artifacts
3. HTML reports are generated automatically
4. No credit card required - completely FREE!

## View Results
1. Go to your GitHub repository
2. Click "Actions" tab
3. Click on latest workflow run
4. Download "monitor-results" artifact
5. Extract and open HTML file in browser

## Manual Trigger
You can also run manually:
1. Go to "Actions" tab
2. Select "Morocco Website Monitor"
3. Click "Run workflow"
